name: Projeto em português com exceções
description: Código deve ser em português, mas termos técnicos consagrados (refresh, logout, login, accessToken, refreshToken, JWT) não devem ser traduzidos.
type: feedback
---

Manter o código em português pois alguns desenvolvedores não sabem inglês.

**Why:** Decisão da equipe para acessibilidade dos devs.

**How to apply:** Traduzir nomes de funções, variáveis e mensagens de erro para português. Manter em inglês termos técnicos já consagrados como refresh, logout, login, accessToken, refreshToken, JWT, routes, router, service, repository, controller, etc. Isso se aplica também a nomes de arquivo (ex: `auth.service.js`, `usuario.repository.js`).
